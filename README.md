## 노트

DP 팁: 부분공간을 하나씩 늘려가면서 그 때 고려해야 하는 상황이 무엇인지 파악. 그 상황을 확정하냐 배제하느냐 따라 DP를 진행할 수 있다.
- 확정할 때: 그 상황을 제거함. 그러면 이전에 고려한 부분공간으로 문제 범위가 줄어듦 -> 이미 DP 테이블을 채웠으므로 최적의 값 앎
- 배제할 때: 그 상황을 고려할 필요가 없기 때문에 바로 직전까지 채웠던 DP 테이블의 최적 값만 고려하면 됨
즉, dp[i] = max(i번째 상황을 확정할 때 최적 값, i번째를 배제할 때 최적 값) -> max(dp[i-X] + f(i), dp[i-1]) , X: i번째 상황을 고려함으로써 제거되는 문제 범위, f(i): i번째 고려할 때 얻는 점수 혹은 무언가.
핵심은 **부분문제의 범위를 넓혀가면서 그때그때 상황을 확정하거나, 배제함으로써 dp[i]를 구할 때 이미 구해놨던 dp 값들을(dp[i-X], dp[i-1]) 이용할 수 있다**는 것이다.
참고: https://www.acmicpc.net/problem/2616 기차의 수를 늘리고, 고려할 객실의 수를 하나씩 늘려가면서 고려할 상황이 뭐가 추가되는지 확인 -> 그 상황을 확정 or 배제했을 때 각각 최적의 값 도출해서 dp 테이블 채움
