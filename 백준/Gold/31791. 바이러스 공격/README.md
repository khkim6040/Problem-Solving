### 출력 

 <p>도시 내에 안전한 구역이 없다면 <code><span style="color:#e74c3c;">-1</span></code>을 출력한다.</p>

<p>도시 내에 안전한 구역이 있다면 안전한 구역들의 좌표를 행 번호가 증가하는 순으로, 행 번호가 같다면 열 번호가 증가하는 순으로 한 줄에 하나씩 출력한다.</p>
